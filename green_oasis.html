Green Oasis

E-commerce Website Template for Seliing Plants

Project Description:

A simple, user-friendly e-commerce platform designed to sell a variety of plants. This project demonstrates proficiency in HTML, CSS, and JavaScript, with a focus on creating an aesthetically pleasing and functional user interface.

Key Features:

  • Responsive Design: Optimized for different devices and screen sizes.
  • Clean Aesthetic: Utilizes a color scheme of black, grey, white, and various shades of green for a modern and refreshing look.
  • User-friendly Navigation: Simple and intuitive navigation ensuring a seamless user experience.
  • Interactive Elements: Includes interactive elements like product carousels, hover effects, and dropdown menus.
  • Shopping Cart: Functional shopping cart allowing users to add, remove, and view items.
  • Product Filtering: Options to filter plants by category, size, and price.

    • Technologies Used:

  • HTML: Structured the content and layout of the website.
  • CSS: Styled the website, incorporating flexbox and grid layouts for a responsive design.
  • JavaScript: Added interactivity and dynamic content

    Challenges Faced and Solutions:

  • Responsive Design: Ensuring the website looked good on all devices was a challenge. Used CSS media queries to adapt the layout for different screen sizes.
  • Cross-browser Compatibility: Ensured the website worked seamlessly across different browsers by testing and using vendor prefixes where necessary.
